# Examining variability and generalization in dimension-based statistical learning for speech

Analysis code and data for:

Steffman, J., He, M., & Segger-Staveley, S. (2025). Examining variability and 
generalization in dimension-based statistical learning for speech: The case of 
place of articulation. *JASA Express Letters*, 5(6).

Originally hosted on [OSF](https://osf.io/3zk5a/).

## My Contribution

I contributed to stimulus design, experimental implementation, data analysis, 
and writing up of results as a Research Assistant on this project (PI: Dr Jeremy 
Steffman, University of Edinburgh).

## Methods

- **Language:** R
- **Statistical analysis:** Mixed-effects regression (lme4), Bayesian inference (brms)
- **Data processing:** tidyverse, dplyr
- **Visualisation:** ggplot2
